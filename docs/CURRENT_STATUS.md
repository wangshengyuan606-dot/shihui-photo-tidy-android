# 当前状态与边界

更新时间：2026-06-26

## 当前状态

项目当前是 Android 真机可运行的高保真 Demo，主要用于作品集展示和 vibe coding 流程复盘。当前代码已经具备完整页面框架、真实相册读取、局部智能搜索、相似照片识别、App 内回收站和压缩页 UI。

## 已完成能力

- Kotlin + Jetpack Compose Android 工程。
- 首页 / 整理 / 我的三 Tab。
- 权限说明页。
- Android 13+ 图片和视频权限请求。
- Android 14+ 部分照片授权兼容。
- MediaStore 读取真实照片和视频。
- 首页照片盲盒。
- 首页搜索框。
- 搜索支持文件名、相册名、日期、媒体类型、用户标签。
- ML Kit 图片标签接入。
- ML Kit 人脸检测接入。
- EXIF GPS 地点索引。
- 搜索索引本地缓存。
- 日期 / 相册 / 相似三种整理模式。
- 月份详情日历视图。
- 月份详情照片堆叠视图。
- 大图滑动保留 / 删除。
- 标签归档。
- 手势习惯设置。
- App 内回收站。
- 回收站恢复。
- 回收站彻底删除确认。
- 压缩选择页 UI。
- Debug APK 构建和真机安装验证。

## 当前缺陷

- 整理状态主要保存在内存中，App 重启后不能完整保留全部整理进度。
- 回收站彻底删除不是真实系统相册删除。
- 压缩页未执行真实图片或视频压缩。
- 人物能力只做到人脸检测，未做人脸聚类和命名闭环。
- 大型相册下仍缺少完整分页、后台扫描和任务进度管理。
- 部分 UI 仍以作品集 Demo 效果为优先，未完成多机型系统性适配。
- 当前 APK 是 Debug 包，不是发布签名包。

## 安全边界

- App 默认本地处理照片，不上传服务器。
- 删除流程当前只改变 App 内状态，避免 Demo 阶段误删用户原始照片。
- 如果后续接入真实删除，必须使用 Android 系统确认流程，并明确告知用户风险。
